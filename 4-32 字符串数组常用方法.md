# 字符串数组常用方法

* 1. 字符串 format方法

```
if(!String.prototype.format ){
    String.prototype.format = function() {
        var e = arguments;
        return this.replace(/{(\d+)}/g,function(t, n) {
            return typeof e[n] != "undefined" ? e[n] : t;
        })
    };
}

var template = "今天是{0},马上就是{1}！";
console.log(template.format("2017","2018"));
```
