# 在总的列表数组对象中查找返回的对象中存在的字段

* 一种方法在总数组中遍历，存在值的数组中查找
  ```
    var list1 = ['游泳',"跳舞"];
    var list2 = [{name:"游泳"},{name:"跳舞"},{name:"舞"},{name:"跳"}];

    list2.forEach(function(el,key){
        if(list1.indexOf(el.name) != -1){
          el.select = true;
        }
    })
  ```
